## Installation
- Clone the repository
```bash
  git clone git@github.com:CyberGA/dairylog.git
```
- Go to the project folder

```bash
  cd dairylog
```
- Install dependencies
```bash
  npm install
```

- Run the project
```bash
  npm run dev
```
